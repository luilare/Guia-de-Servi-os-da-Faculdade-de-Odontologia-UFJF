# Guia de Serviços Odontológicos - UFJF

Conteúdo:
- `index.html` — página principal pronta.
- `style.css` — estilos customizados (usa Tailwind CDN + CSS custom).
- `README.md` — este arquivo.

## Como testar localmente
1. Abra a pasta `guia-odontologia-ufjf-site`.
2. Clique duas vezes em `index.html` ou abra com o navegador.
3. (Opcional) Para desenvolvimento com recarga automática, use VS Code + extensão *Live Server*.

## Como publicar no GitHub Pages
1. Crie um repositório no GitHub (público).
2. Adicione os arquivos desta pasta ao repositório (`index.html` deve ficar na root).
3. Vá em **Settings → Pages** e configure a branch `main` e a pasta `/ (root)`.
4. Aguarde a publicação — o link será `https://<seu-usuario>.github.io/<nome-do-repo>/`.

## Dicas rápidas de edição
- Para alterar as cores principais, edite as variáveis em `style.css`.
- Para trocar logos, substitua as URLs das imagens no `index.html`.
- Para adicionar novas seções, copie uma `<section id="...">...</section>` existente.
